# Publish Checklist

Before publishing, confirm:

- [ ] Date is correct
- [ ] Game start times checked
- [ ] Probable pitchers checked
- [ ] Odds checked within acceptable time window
- [ ] Internal price checks passed
- [ ] Same-game market conflicts checked
- [ ] No opposing moneyline/run-line contradictions on the official card
- [ ] Lineups checked or uncertainty flagged
- [ ] Injuries/news checked
- [ ] Weather checked
- [ ] Bullpen usage checked
- [ ] Edge score included
- [ ] Market implied probability included
- [ ] LyDia projected probability included
- [ ] Raw edge included
- [ ] Competitor logic is summarized, not copied
- [ ] Verifier status included
- [ ] Held/rejected picks are not presented as official
- [ ] Banned gambling language removed
- [ ] "Playable to" and "Pass at" are not shown in public/paid copy
- [ ] Responsible gambling footer included

If any required item fails, hold the pick.
