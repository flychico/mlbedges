# Data Sources

Use multiple sources. Never rely on one page when pitcher, odds, or injury status is uncertain.

## Required categories

### Probable pitchers

- MLB official probable pitchers
- Team beat reports when needed
- FanGraphs/RosterResource when useful

### Odds

- At least one sportsbook line
- Prefer consensus or multiple books when available
- Record opening, current, and closing line when possible

### Player/team stats

- FanGraphs
- Baseball Savant / Statcast
- MLB official stats

### Injuries/news

- MLB team pages
- RosterResource
- beat writers / team news
- sportsbook preview pages only as secondary support

### Weather

- Weather forecast near stadium
- Radar close to first pitch for risky games

### Competitors

- Public sports betting previews
- Sportsbook research pages
- Picks sites
- Social posts/articles when accessible

## Source reliability levels

```text
A = official league/team/stat source or direct sportsbook odds
B = respected analytics/stat site
C = public betting preview with cited data
D = social/media opinion without clear data
```

## Rule

Use low-reliability sources for context, not final claims.
