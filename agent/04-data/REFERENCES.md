# Methodology References

These are useful grounding references for LyDia Agent.

## Pitching stats

FanGraphs Library explains FIP as a way to estimate pitcher performance independent of factors outside the pitcher's control that affect runs allowed. FanGraphs also defines xFIP and SIERA as pitcher skill indicators that go beyond raw ERA.

- https://library.fangraphs.com/pitching/fip/
- https://library.fangraphs.com/pitching/xfip/
- https://library.fangraphs.com/pitching/siera/

## Expected stats

Baseball Savant/Statcast defines xwOBA using exit velocity, launch angle, and sprint speed on certain batted balls. Expected stats help separate contact quality from box-score outcomes.

- https://baseballsavant.mlb.com/leaderboard/expected_statistics

## Closing-line value

Closing-line value compares the price taken to the closing market price. It is useful for evaluating whether the process is beating the market, not just whether a pick won.

- https://www.thelines.com/betting/guides/closing-line-value/
- https://oddsjam.com/betting-education/closing-line-value
