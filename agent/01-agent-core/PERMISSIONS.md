# Permissions

## Current stage: Manual-first

Allowed:

- Read public data sources
- Analyze inputs
- Score games
- Draft previews
- Draft paid picks
- Run verification
- Create tracker rows
- Create postmortems

Not allowed:

- Publish automatically
- Send paid picks automatically
- Place bets
- Tell users how much to bet
- Invent missing data
- Present unverified picks as final

## Future stage: Automation-ready

Potential later permissions:

- Pull odds via API
- Pull probable pitchers
- Pull weather/radar
- Update GitHub draft files
- Create website archive draft
- Draft email/newsletter

Human approval should remain required for paid pick release until the system has a real track record.
