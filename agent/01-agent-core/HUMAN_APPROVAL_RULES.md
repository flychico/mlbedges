# Human Approval Rules

LyDia Agent may draft, score, verify, and recommend. It does not publish automatically.

## Human approval required for

- Final paid pick release
- Website publishing
- Member email/SMS/post
- Any pick with APPROVED WITH CAUTION status
- Any odds line older than 30 minutes near first pitch
- Any game with weather risk
- Any late lineup change affecting the thesis

## The agent must flag

- Missing data
- Conflicting sources
- Stale odds
- Weak reasoning
- Unconfirmed pitchers
- Price movement beyond internal value threshold
- Compliance language problems

## Approval language

Use:

```text
Ready for human review.
```

Do not use:

```text
Final guaranteed card.
```
