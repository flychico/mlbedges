Pitcher Matchup Tool

Every probable starter on the card, translated: which numbers are real skill, which are luck waiting to correct, and how deep each arm takes you before the bullpen decides the rest.

How to read it: the big number is the LyDia Pitcher Score (0–100) — run prevention, WHIP, K/BB profile, and sample size in one grade. Tier labels translate the decimals: every ~0.50 of ERA and ~0.10 of WHIP is roughly one class of pitcher. K-BB% is the most skill-driven stat on the card; BABIP far from ~.300 flags luck that usually corrects; IP/start is how much of the game the bullpen inherits.
