Totals Projections

How many runs should tonight's games produce? Each projection blends both lineups' scoring, the opposing starters' skill (FIP-based), the ballpark's run environment, and bullpen condition — shown next to the market's over/under line.

How to read it: the projection is a median expectation — real games land within ±2.5 runs of it routinely, so only lean when the gap to the line is 0.5+ runs, and treat leans as research, not picks. Lines are captured each morning and kept all day (re-captured only if a starter changes). Every projection is graded nightly against the final score on the learning ledger — this tool earns trust or loses it in public.
