Offense Matchup Tool

Which lineup is hot, which is actually good, and how each hits the hand of tonight's opposing starter. Quality and heat are different things — this tool shows both.

7d 15d 30d
How to read it: OPS (window) is quality right now; Δ vs season is heat — a bad offense can be hot and an elite one can be cold. Tier labels translate the decimals: every ~25 OPS points is one class of offense (.700 below avg · .750 solid · .800 good · .850 All-Star · .900 elite). HOW a team is hot matters: contact-driven heat holds up; strikeout-heavy power binges collapse. The vs-hand line is each lineup's full-season OPS against tonight's opposing starter's throwing hand.
